- Linux system
- Bash 4.0+
- Core utilities: `top`, `df`, `free`, `awk`

## Installation

1. Clone repository:
```bash
git clone https://github.com/chenxing-dev/system-health.git
cd system-health
```

2. Make script executable:
```bash
chmod +x system_health.sh
```
